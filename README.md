# Rolling Shells
Login to Unprotected Shells Randomly Using SHODAN

# Intoduction
A simple tool that uses shodan api to search for open shell and save it to use later randomly.

# Installation

``cmd
git clone https://github.com/DeyaaMuhammad/rollingshells
``
